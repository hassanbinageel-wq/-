# 🌙 منبّه الصلاة — Salah Alarm

منبّه صلاة بتصميم فاخر (أسود وذهبي، RTL عربي) — **ما يطفّى المنبّه إلا لمّا تصوّر المغسلة أو الصنبور بالكاميرا**.
الفكرة: تضمن إنك قمت فعليًا وتوضّيت قبل ما يسكت المنبّه.

تطبيق **أندرويد أصلي (Kotlin)** كامل وقابل للتعديل، جاهز للبناء.

---

## ✨ المزايا

- ⏰ منبّهات متعددة بأوقات وأسماء (الفجر، الظهر…).
- 📅 تكرار أسبوعي (اختَر الأيام) أو تاريخ مرّة وحدة.
- 🔊 اختيار صوت المنبّه من أصوات الجهاز (أو ضع أذان خاص بك).
- 📸 **الإيقاف بالكاميرا فقط:** التطبيق يستخدم تعرّفًا ذكيًا داخل الجهاز (ML Kit) يكشف المغسلة/الصنبور — وأول ما يشوفه، يسكت المنبّه فورًا.
- 🔒 يشتغل فوق شاشة القفل، ويهتزّ، ويرنّ بصوت المنبّه حتى لو الجوال صامت.
- 🌙 واجهة عربية فاخرة (أسود/ذهبي) بهوية `@hsn.pmt`.
- ⚡ يعمل بدون إنترنت (نموذج التعرّف مدمج).

## 📦 طريقة الحصول على ملف APK

> ليش ما فيه APK جاهز داخل المجلد؟ لأن بناء أي تطبيق أندرويد يحتاج تنزيل Android SDK ومكتبات Google، وهذي ما تُنزّل في البيئة اللي تولّد فيها الكود. الحل أسهل مما تتوقع: عندك طريقتين، وكلتيهما تعطيك APK.

### الطريقة 1 — GitHub Actions (الأسهل، بدون تنصيب أي شي على جهازك) ✅

1. سوِّ مستودع جديد على GitHub.
2. ارفع محتويات هذا المجلد كامل (فيه ملف `.github/workflows/build-apk.yml`).
3. روح تبويب **Actions** في المستودع — البناء يبدأ تلقائيًا (أو اضغط **Run workflow**).
4. بعد ما يخلص (دقايق)، افتح نتيجة البناء وحمّل الملف من **Artifacts → `salah-alarm-apk`**.
5. فكّ الضغط، وبتلقى `app-debug.apk`. انقله للجوال وثبّته (فعّل «تثبيت من مصادر غير معروفة»).

### الطريقة 2 — Android Studio (نقرة وحدة)

1. ثبّت [Android Studio](https://developer.android.com/studio).
2. **File → Open** واختَر هذا المجلد. خلّه يسوّي Gradle Sync (ينزّل المكتبات أول مرة).
3. **Build → Build Bundle(s) / APK(s) → Build APK(s)**.
4. لمّا يخلص، اضغط **locate** وبتلقى `app-debug.apk` جاهز.
   - أو وصّل جوالك واضغط ▶ Run عشان يثبّته مباشرة.

## 🔐 الأذونات المطلوبة (وافق عليها أول تشغيل)

- **الكاميرا** — ضرورية لإيقاف المنبّه بتصوير المغسلة.
- **التنبيهات والتذكيرات (Exact Alarm)** — عشان يرنّ بالضبط في وقته. التطبيق بيطلب منك تفعيلها أول مرة.
- **الإشعارات** — لعرض المنبّه فوق الشاشة (أندرويد 13+).
- العرض فوق شاشة القفل والاهتزاز — مفعّلة تلقائيًا.

> نصيحة: من إعدادات النظام، اسمح للتطبيق بالعمل في الخلفية / استثنِه من موفّر البطارية، عشان ما يتأخر المنبّه.

## 🛠️ التخصيص

- **أذان خاص:** حُط ملف `adhan.mp3` داخل `app/src/main/res/raw/` ثم اختره من «صوت التنبيه» داخل التطبيق.
- **حسّاسية التعرّف والكلمات:** عدّل `DETECT_KEYWORDS` و `CONFIDENCE` في
  `app/src/main/java/com/hsnpmt/salahalarm/Util.kt`.
  - ارفع `CONFIDENCE` (مثلاً `0.7f`) لتقليل الكشف الخاطئ، أو نزّلها لو ما يكشف بسرعة.
- **الألوان والهوية:** `app/src/main/res/values/colors.xml`.
- **اسم التطبيق:** `app/src/main/res/values/strings.xml` (`app_name`).

## 🧠 كيف يشتغل الكشف؟

عند رنين المنبّه، الشاشة الوحيدة اللي تطلع فيها زرّ كاميرا كبير. تفتح الكاميرا، والتطبيق يحلّل كل لقطة لحظيًا عبر **ML Kit Image Labeling** (نموذج مدمج يعمل أوفلاين). إذا طلع وسم ضمن الكلمات (`sink`, `tap`, `faucet`…) بثقة كافية — يرسل النتيجة ويسكت المنبّه على طول. ما فيه طريقة ثانية توقفه (حتى زر الرجوع معطّل في شاشة المنبّه).

## 🧩 البنية التقنية

| العنصر | التفصيل |
|---|---|
| اللغة | Kotlin |
| الواجهات | XML Views + viewBinding |
| الكاميرا | CameraX 1.3.4 |
| التعرّف | ML Kit Image Labeling 17.0.9 (أوفلاين) |
| الجدولة | AlarmManager `setAlarmClock` (تنبيه دقيق) |
| أقل إصدار | Android 8.0 (API 26) |
| الهدف | Android 14 (API 34) |
| AGP / Gradle / Kotlin | 8.5.2 / 8.7 / 1.9.24 |

### أهم الملفات
```
app/src/main/java/com/hsnpmt/salahalarm/
├── MainActivity.kt        # قائمة المنبّهات + اختصار الكاميرا
├── EditAlarmActivity.kt   # إضافة/تعديل منبّه (وقت، تاريخ، أيام، صوت)
├── AlarmActivity.kt       # شاشة الرنين فوق القفل (الإيقاف بالكاميرا فقط)
├── CameraActivity.kt      # CameraX + كشف المغسلة/الصنبور
├── AlarmService.kt        # خدمة أمامية: صوت + اهتزاز + إشعار شاشة كاملة
├── AlarmScheduler.kt      # حساب وقت الرنين والجدولة
├── AlarmReceiver.kt       # يستقبل وقت المنبّه ويشغّل الخدمة
├── BootReceiver.kt        # يعيد جدولة المنبّهات بعد إعادة التشغيل
├── AlarmStore.kt          # حفظ المنبّهات (SharedPreferences + JSON)
├── AlarmAdapter.kt        # عناصر القائمة
├── Alarm.kt               # نموذج البيانات
└── Util.kt                # الثوابت + تنسيق الوقت/الأيام + إعدادات الكشف
```

## 📤 للنشر على Google Play (لاحقًا)

ملف الإخراج هنا `app-debug.apk` (للتجربة). للنشر الرسمي تحتاج:
1. بناء `release` موقّع: **Build → Generate Signed Bundle / APK**، وأنشئ Keystore واحفظه بأمان.
2. ارفع ملف `.aab` على Google Play Console.
